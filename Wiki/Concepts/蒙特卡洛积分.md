# 蒙特卡洛积分 (Monte Carlo Integration)

## 定义

蒙特卡洛积分是一种使用随机采样来近似计算积分值的数值方法。其收敛率与被积函数的维数无关，是渲染中高维积分（光传输方程、参与介质散射、相机传感器响应等）的唯一实用方法。

随机算法分两类：**Las Vegas**（随机化但结果确定，如 Quicksort 随机选主元）和 **Monte Carlo**（结果依赖随机数，平均正确）。MC 积分属于后者。

## 概率基础

| 概念 | 定义 | 要点 |
|------|------|------|
| **随机变量** | 由随机过程选取的值 | 离散或连续域 |
| **PMF / PDF** | 概率质量函数 / 概率密度函数 | PDF ≥ 0 且积分为 1；PDF 值可 > 1 |
| **CDF** | 累积分布函数 P(X ≤ x) | CDF 是 PDF 的积分 |
| **期望值** | E[f(X)] = ∫ f(x)·p(x) dx | 线性性质：E[af] = aE[f]，E[f+g] = E[f]+E[g] |
| **方差** | Var[X] = E[X²] − (E[X])² | 方差具有可加性（独立变量） |
| **ξ (xi)** | 正则均匀随机变量 ∈ [0,1) | 所有采样的基础 |

## 核心公式

**均匀采样估计量**（区间 [a,b]）：

> F_n = (b−a)/n · Σᵢ f(Xᵢ)，其中 Xᵢ ~ Uniform[a,b]

**一般 MC 估计量**（任意 PDF p(x)）：

> F_n = (1/n) Σᵢ f(Xᵢ) / p(Xᵢ)

**性质**：E[F_n] = ∫f(x)dx（**无偏**），唯一要求 p(x) > 0 wherever f(x) ≠ 0

## 误差特性

- 方差随 n **线性减少**
- 误差以 **O(n^{−1/2})** 速度下降（4× 样本 → 2× 精度）
- 收敛率与维数无关（传统 quadrature 随维度指数恶化）
- 效率 ε = 1 / (Var × Time)
- **样本方差**可用 Bessel 校正估计：s² = Σ(Xᵢ − X̄)² / (n−1)
- **有偏估计量**若 bias → 0 则称为**一致的**(consistent)；MSE = Var + Bias²

## 方差减少技术

### 1. 分层采样 (Stratified Sampling)

将积分域分割为非重叠层（strata），每层内独立采样。**永不增加方差**——只有当所有层均值相等时方差不减。分层应尽量使各层均值差异大。

缺点：受"维数灾难"影响——d 维 s 层需 s^d 个样本。实践中可独立分层各维度再随机关联。

### 2. 重要性采样 (Importance Sampling)

选择 p(x) ≈ |f(x)| 的采样分布，使 f/p 尽量均匀。

- **理想 PDF** p*(x) = |f(x)| / ∫|f(x)|dx → 方差为零（但需已知积分值）
- **好的近似 PDF** 可带来数量级的方差减少（例如高斯函数示例中 6.7× 改善）
- **差的 PDF** 会**增加方差**——在被积函数大处概率低，f/p 的峰值被放大

### 3. 多重重要性采样 (MIS)

处理乘积被积函数 f(x) = g(x)·h(x) 的利器。结合多个采样策略，用**权重函数**消除单一策略不匹配导致的方差尖峰。

- **平衡启发式** (balance heuristic)：wᵢ(x) = nᵢ·pᵢ(x) / Σⱼ nⱼ·pⱼ(x)
- **幂启发式** (power heuristic, β=2)：进一步压制低概率贡献，实践中通常更优
- **单样本模型**：选一个策略采样，平衡启发式可证最优
- **MIS 补偿**：将某些 PDF 的低概率区域置零→避免过采样低值区域

### 4. 俄罗斯轮盘赌 (Russian Roulette)

以概率 q 跳过低贡献样本（用常数 c 替代），以概率 1−q 继续评估并除以 1−q 补偿。期望值不变（无偏），但**总是增加方差**；通过跳过昂贵计算提高**效率**。

### 5. 分割 (Splitting)

在多维积分中，对某些维度取更多样本，分摊另一维度的昂贵计算。例如：每个像素样本射出多条光照采样光线，分摊追踪相机光线的成本。

## 采样方法

### 反函数采样法 (Inversion Method)

1. 求 PDF 的 CDF：P(x) = ∫_{-∞}^{x} p(t) dt
2. 取均匀随机数 ξ ∈ [0,1)
3. 求解 x = P⁻¹(ξ)

**离散情形**：在 CDF 阶梯函数中查找 ξ 所在区间（`SampleDiscrete`）
**连续情形**：解析求逆 CDF（如线性函数的二次方程求根）

### 多维分布变换

- **1D 变换**：若 Y = T(X)，则 p_Y(y) = p_X(T⁻¹(y)) / |T'(T⁻¹(y))|
- **多维变换**：p_Y(y) = p_X(T⁻¹(y)) / |det(J_T)|，J_T 为 Jacobian 矩阵
- **极坐标变换**：p(r,θ) = r · p(x,y)
- **球坐标变换**：p(θ,φ) = sinθ · p(ω)
- **条件采样**：先采样边缘分布 p(y) = ∫p(x,y)dx，再采样条件分布 p(x|y)
- **双线性采样**：通过两次线性采样组合实现 2D 双线性函数的精确采样

## 历史

- 由 **Stanislaw Ulam** 和 **John von Neumann** 于 1947 年引入
- Enrico Fermi 独立发明
- Veach (1997) 奠定了在渲染中应用的理论基础（MIS、双向路径追踪、MLT）
