# 纹理与材质 (Textures and Materials)

## 定义

[[反射模型]]只描述光在表面一点如何散射，而**纹理与材质**系统解决的是：在表面的每一点，**使用哪个** BSDF、其**参数是什么**。

pbrt 将这一问题分为两个组件：

1. **纹理 (Texture)** — 将表面上的点映射到值（标量或光谱）的函数
2. **材质 (Material)** — 在表面各点评估纹理，确定 BSDF 参数

分离纹理和材质使得它们可以**任意组合**——同一套纹理可以用于不同材质，同一种材质可以用不同纹理参数化。

## 纹理

纹理在 pbrt 中是一个非常通用的概念：**从某个域（如参数空间、物体空间）到另一个域（如光谱、实数）的函数**。

### 纹理类型

| 类型 | 描述 |
|------|------|
| **常数纹理** | 0 维——返回固定值 |
| **图像纹理** | 2D 像素数组，最常用 |
| **程序化纹理** | 用数学函数计算（棋盘格、噪声等） |
| **组合纹理** | 基于其他纹理的值计算 |

### 纹理反走样

纹理是图像走样的重要来源（如远处的棋盘格）。解决方法是让纹理函数根据**采样率自适应调整频率内容**：

- **MIP 映射 (MIP Map)**：预计算多级分辨率的图像金字塔
- **EWA 滤波**：椭圆加权平均，根据纹理空间中的采样足迹进行各向异性滤波
- **光线微分 (Ray Differentials)**：跟踪相邻像素光线在纹理空间中的足迹，估计所需滤波宽度

纹理预滤波比增加像素采样率**高效得多**——用一个样本就能得到反走样结果。

### 纹理坐标生成

将 3D 表面点映射到 2D 纹理坐标 (u, v) 的过程：

- **(u, v) 参数化**：使用 Shape 自身的参数坐标
- **球面/圆柱映射**：数学投影
- **平面投影**：正交/透视投影
- **3D 纹理坐标**：直接使用物体空间点（适用于程序化纹理，如 Perlin 噪声）

## 材质

材质是纹理和 BSDF 之间的**桥梁**。`Material` 接口定义了 `GetBSDF()` 方法，返回特定表面点的 BSDF。

### pbrt 中的材质实现

| 材质 | 描述 | 关联反射模型 |
|------|------|-------------|
| **DiffuseMaterial** | 漫反射 | Lambertian |
| **DielectricMaterial** | 光滑或粗糙玻璃 | Dielectric BSDF |
| **ConductorMaterial** | 金属 | Conductor BRDF |
| **CoatedDiffuseMaterial** | 涂层漫反射 | LayeredBxDF |
| **CoatedConductorMaterial** | 涂层金属 | LayeredBxDF |
| **SubsurfaceMaterial** | 次表面散射 | BSSRDF |
| **ThinDielectricMaterial** | 薄透明层 | 简化介质 |
| **HairMaterial** | 头发 | Hair BSDF |
| **MeasuredMaterial** | 测量数据 | Measured BSDF |
| **MixMaterial** | 混合两种材质 | 加权组合 |
| **DiffuseTransmissionMaterial** | 漫透射 | 双侧漫反射 |

### 法线贴图与凹凸贴图

材质可以通过修改表面法向量来模拟细节几何，而无需增加实际的几何复杂度：

- **凹凸贴图 (Bump Mapping)**：用标量位移函数扰动法向量
- **法线贴图 (Normal Mapping)**：直接指定每点的法向量
