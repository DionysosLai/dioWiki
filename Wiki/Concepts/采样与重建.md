# 采样与重建 (Sampling and Reconstruction)

## 定义

采样与重建是将连续信号转换为离散表示、再从离散样本恢复连续信号的过程。在渲染中，入射辐射亮度是定义在胶片平面上的**连续函数**，但最终输出是离散的像素网格。这个转换过程的质量直接决定了渲染图像的品质——处理不当会产生走样（aliasing）伪影，处理得当则能以少量额外计算大幅提升图像质量。

## 傅里叶分析基础

采样理论的数学基础是**傅里叶分析**——将函数分解为正弦波加权和的理论。

### 傅里叶变换

任何（满足条件的）函数都可以在**空间域**和**频域**之间转换：

- **傅里叶变换**：空间域 → 频域（分析函数的频率组成）
- **逆傅里叶变换**：频域 → 空间域

重要的傅里叶变换对：

| 空间域 | 频域 |
|--------|------|
| Box（方波） | Sinc |
| Gaussian（高斯） | Gaussian |
| 常数 | Delta 函数 |
| 正弦波 | 平移 delta |
| Shah（脉冲列） | Shah |

### 卷积定理

- 空间域中的**乘法** = 频域中的**卷积**
- 空间域中的**卷积** = 频域中的**乘法**

这一对偶关系是理解采样与重建的关键。

## 理想采样与重建

### 采样过程

采样在数学上等价于将连续函数乘以 **Shah 函数**（脉冲列），周期 T 定义采样率。在频域中，这对应于将原始频谱以 1/T 为周期无限复制。

### 理想重建

用频域中的 **Box 函数**截取原点处的频谱副本，丢弃其余副本。在空间域中，这对应于与 **sinc 函数**做卷积——这是理论上完美的重建滤波器。

缺点：sinc 函数具有**无限支撑**（infinite extent），实际中无法使用所有样本来计算重建值。

## 走样 (Aliasing)

当信号不是带限的，或采样率不够高时，频谱副本**重叠**，高频细节泄漏到低频区域，产生走样。

### 采样定理 (Nyquist)

> 如果均匀采样频率大于信号最高频率的**两倍**（Nyquist 频率），则可以完美重建原信号。

在光线追踪中，场景图像函数通常**不是带限的**（存在硬边缘、高频纹理），因此完美重建在任何采样率下都不可能。

### 反走样策略

1. **增加采样率** — 最直接但最昂贵
2. **非均匀采样** — 将走样的规则图案转化为噪声（视觉上更不易察觉）
3. **纹理预滤波** — 在采样前降低信号频率（见 [[纹理与材质]]）
4. **自适应采样** — 在高变化区域增加样本

## 六种采样器实现

pbrt 实现了从简单到先进的六种采样器：

| 采样器 | 特点 | 适用场景 |
|--------|------|---------|
| **Independent** | 完全随机，无相关性 | 基线参考 |
| **Stratified** | 分层抖动（jittered），保证覆盖 | 低维问题 |
| **Halton** | 基于互质基数的低偏差序列 | 通用，渐进式渲染 |
| **Sobol'** | (0,2)-序列，最优 2D 分层 | 高质量积分 |
| **Paddded Sobol'** | Sobol' + 随机填充，避免像素间相关 | 生产渲染 |
| **Z-Sobol'** | Morton 曲线排列的 Sobol' | GPU 友好 |

### 低偏差序列

低偏差序列（也称**准随机序列**）是一类确定性点集，在高维空间中比随机点更均匀地分布。它们是**准蒙特卡洛 (QMC)** 方法的基础，在渲染中通常比纯随机采样收敛更快。

- **Halton 序列**：基于 van der Corput 序列，不同维度使用不同质数基
- **Sobol' 序列**：基于生成矩阵，具有最优的 (0,2)-序列性质
- **偏差 (Discrepancy)**：衡量点集均匀性的数学度量

## 图像重建滤波器

采样完成后，需要用**重建滤波器**将多个样本混合为最终像素值。pbrt 实现了五种滤波器：

| 滤波器 | 特点 |
|--------|------|
| **Box** | 最简单但质量最差，等权平均 |
| **Triangle** | 线性衰减，比 Box 好 |
| **Gaussian** | 平滑，稍有模糊 |
| **Mitchell** | 参数可调的三次多项式，平衡锐利与振铃 |
| **Lanczos (Windowed Sinc)** | 截断的 sinc，理论最优的实用近似 |

重建是卷积运算：每个样本对周围像素的贡献由滤波器权重决定。

## 采样与积分的关系

在 pbrt 中，**积分**通常比重建更为重要。每个像素值本质上是一个积分：

> 像素值 = ∫∫ L(x,y) · f(x,y) · dA

其中 L 是入射辐射亮度，f 是像素滤波器。[[蒙特卡洛积分]]是求解这个积分的方法，采样器决定了积分点的分布。

傅里叶分析也为蒙特卡洛积分的误差提供了洞见——好的采样模式在频域中的能量分布应避免与被积函数的频谱重叠。
