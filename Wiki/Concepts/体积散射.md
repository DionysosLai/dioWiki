# 体积散射 (Volume Scattering)

## 定义

体积散射描述光穿过**参与介质**（participating media）时如何被影响。当场景中不再假设是"真空中的表面集合"时，分布在 3D 空间中的大量微小粒子会吸收、散射和发射光线。这使得渲染可以模拟雾、烟、大气散射、云层中的光线、浑浊水体以及**次表面散射**（光从固体一个位置进入、另一个位置射出）。

## 三大物理过程

| 过程 | 效果 | 系数 |
|------|------|------|
| **吸收 (Absorption)** | 光能被介质转化为其他能量形式（如热） | σₐ |
| **外散射 (Out-Scattering)** | 光被散射到其他方向 | σₛ |
| **内散射 (In-Scattering)** | 来自其他方向的光被散射到当前方向 | σₛ |

**消光系数** σₜ = σₐ + σₛ，描述光沿直线传播时的总衰减率。

**反照率 (Albedo)** ρ = σₛ / σₜ，衡量散射与吸收的比例——高反照率介质看起来明亮（云），低反照率介质看起来暗淡（烟）。

## 透射率 (Transmittance)

描述光从一点传播到另一点时的衰减比例：

> T(p → p') = exp(−∫ σₜ(t) dt)

- **均匀介质**：T = exp(−σₜ · d)（Beer-Lambert 定律）
- **非均匀介质**：需要数值积分（null scattering 技术）

## 相位函数 (Phase Function)

体积散射中的 "BSDF"——描述光在介质中一点被散射时的方向分布。

- **各向同性**：p(ω, ω') = 1/(4π)，向所有方向均匀散射
- **Henyey-Greenstein**：单参数 g ∈ (−1, 1) 控制前向/后向散射偏好
  - g > 0：前向散射（如云）
  - g = 0：各向同性
  - g < 0：后向散射

与 BSDF 的互易性类似，相位函数也满足 p(ω, ω') = p(ω', ω)。

## Null Scattering

一种数学形式化技巧，通过引入虚拟的 "null" 散射事件（不改变光线方向和能量），将非均匀介质问题转化为均匀介质问题，从而实现无偏采样。这是 pbrt 第4版处理非均匀介质的关键技术。

## Medium 接口

pbrt 的 `Medium` 接口描述参与介质在空间中的散射属性：

| 实现 | 描述 |
|------|------|
| **HomogeneousMedium** | 均匀介质——σₐ、σₛ 处处相同 |
| **GridMedium** | 网格介质——用 3D 体素网格存储空间变化的属性 |
| **RGBGridMedium** | RGB 网格介质——用 RGB 值指定随波长变化的散射属性 |
| **CloudMedium** | 云介质——专用的程序化云密度场 |
| **NanoVDBMedium** | 基于 OpenVDB 的稀疏体积数据 |
| **UniformInfiniteMedium** | 无限均匀介质 |

## 传输方程 (Equation of Transfer)

Chapter 14 推导了完整的**传输方程**——将[[光线追踪|光传输方程]]推广到含参与介质的场景：

> L(p, ω) = ∫ T(p, p') · [σₛ · ∫ p(ω, ω') · L(p', ω') dω' + σₐ · Lₑ] dt + T(p, pₛ) · Lₛ

三项贡献：内散射（其他方向的光散射过来）、发射（介质自发光）、透射的表面辐射。

## 体积积分器

| 积分器 | 描述 |
|--------|------|
| **SimpleVolPathIntegrator** | 基础体积路径追踪——在介质中随机采样散射事件 |
| **VolPathIntegrator** | 完整体积路径追踪——支持 MIS、分层材质 |

## 应用

- **大气散射**：天空蓝色、日落红色（Rayleigh 散射）
- **云与雾**：体积光线效果（god rays）
- **次表面散射**：皮肤、牛奶、大理石、蜡——光在固体内部传播后从不同位置射出
- **光束效果**：光穿过烟雾/灰尘时的可见光柱
- **分层材质**：Chapter 14.3 的 `LayeredBxDF` 用 1D 传输方程模拟涂层、多层漆面
