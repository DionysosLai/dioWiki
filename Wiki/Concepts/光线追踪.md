# 光线追踪 (Ray Tracing)

## 定义

光线追踪是一种通过模拟光线在场景中传播路径来生成图像的渲染算法。其核心操作是沿光线的参数化形式 **r(t) = o + t·d** 追踪，求解光线与场景几何体的交点。光线追踪避开了光的波动性（可见光波长仅几百纳米），在宏观尺度上以粒子/射线模型高效模拟光的传播，这使得它在计算上切实可行。

## 七大核心组件

| 组件 | 功能 | 关键技术 |
|------|------|---------|
| **1. 相机与胶片** | 确定观察位置、成像方式和感光响应 | 针孔模型（eye + image plane）、透镜模型、viewing volume |
| **2. 光线-物体求交** | 计算光线与几何体的交点及表面性质 | 参数化光线代入隐式方程（如球体二次方程求根）、[[BVH加速结构]] O(m log n) |
| **3. 光源分布** | 模拟光能的几何和辐射度分布 | 点光源（1/r² 衰减 + 余弦衰减）、面光源、光的线性性→随机光源采样 |
| **4. 可见性** | 判断光源是否照亮表面点 | 阴影射线（shadow ray）：构建点到光源的光线，检测是否有遮挡 |
| **5. 表面散射** | 描述入射光在表面如何重新分布 | [[BSDF]]（BRDF/BTDF/BSSRDF）→ 材质参数化实现多种外观 |
| **6. 间接光传输** | 模拟多次弹射的全局光照 | 光传输方程 L_o = L_e + ∫ f·L_i·cosθ dω、[[蒙特卡洛积分]]求解 |
| **7. 光线传播** | 描述光穿过参与介质的行为 | 消光（吸收+散射）、透射率、体积光传输方程（烟雾/大气） |

## 光传输方程

光线追踪的数学核心是**光传输方程**（也称渲染方程），由 Kajiya (1986) 提出：

> L_o(p, ω_o) = L_e(p, ω_o) + ∫_{S²} f(p, ω_o, ω_i) · L_i(p, ω_i) · |cosθ_i| dω_i

- **L_o**：从点 p 沿方向 ω_o 出射的辐射亮度（radiance）
- **L_e**：自发光项
- **f**：BSDF
- **L_i**：入射辐射亮度
- 该积分没有解析解（除最简单场景），必须用数值方法（[[蒙特卡洛积分]]）求解

## 历史里程碑

| 年份 | 人物 | 贡献 |
|------|------|------|
| 1968 | Arthur Appel | 首次描述光线追踪用于隐藏面和阴影 |
| 1980 | **Turner Whitted** | 里程碑论文——递归光线追踪（镜面反射/折射） |
| 1981 | Cook & Torrance | 微面元反射模型——首次准确渲染金属 |
| 1984 | Cook, Porter, Carpenter | 分布式光线追踪（运动模糊、景深、面光源软阴影） |
| 1984 | Goral et al. | Radiosity——将热传导有限元方法引入渲染 |
| 1986 | **Kajiya** | [[路径追踪]]——光传输方程的严格数学公式 |
| 1997 | **Veach** | MIS、双向路径追踪、Metropolis 光传输 |
| 2001 | Wald, Slusallek, Benthin | 高度优化的实时光线追踪器 |

## 产业化历程

- **1981**：Jim Blinn 渲染 Voyager 2 飞越土星动画
- **1982-84**：《Star Trek II》《Tron》《The Last Starfighter》引入 CG 视觉特效
- **2001**：Marcos Fajardo 创建 Arnold 渲染器
- **2006**：《怪兽屋》首部全 CG 光线追踪电影（Sony Pictures Imageworks）
- **2013**：Walt Disney 转向 Hyperion 物理渲染器
- **2019-20**：NVIDIA RTX 硬件光线追踪开启实时物理渲染时代

## 在 pbrt 中的地位

光线追踪是 pbrt 的**核心算法基础**。系统的所有组件——几何（Chapter 3/6）、材质（Chapter 9）、光源（Chapter 12）、积分器（Chapter 13-14）——都围绕光线追踪展开。pbrt 使用**参数化光线** r(t) = o + t·d 作为贯穿全书的基本数据结构。

## 在虎书中

虎书 Ch4 是光线追踪的入门章节，从 image-order vs object-order 渲染的对比出发，介绍光线参数化、光线-球体/三角形求交、阴影射线、镜面反射等基础概念。相比 [[pbrt]] 假设读者已有基础，虎书从零开始讲解，是学习光线追踪的推荐起点。Ch13 进一步介绍分布式光线追踪（软阴影、景深、运动模糊）。
