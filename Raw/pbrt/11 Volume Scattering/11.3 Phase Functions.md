## 11.3 Phase Functions

Just as there is a wide variety of BSDF models that describe scattering from surfaces, many phase functions have also been developed. These range from parameterized models (which can be used to fit a function with a small number of parameters to measured data) to analytic models that are based on deriving the scattered radiance distribution that results from particles with known shape and material (e.g., spherical water droplets).

> 正如有各种各样描述表面散射的 BSDF 模型一样，许多相位函数也被开发出来。这些范围从参数化模型（可用于将具有少量参数的函数拟合到测量数据）到基于推导已知形状和材料粒子（例如球形水滴）产生的散射辐射亮度分布的解析模型。

In most naturally occurring media, the phase function is a 1D function of the angle between the two directions and; these phase functions are often written as. Media with this type of phase function are called *isotropic* or *symmetric* because their response to incident illumination is (locally) invariant under rotations. In addition to being normalized, an important property of naturally occurring phase functions is that they are *reciprocal*: the two directions can be interchanged and the phase function's value remains unchanged. Note that symmetric phase functions are trivially reciprocal because.

> 在大多数自然存在的介质中，相位函数是两个方向之间角度的一维函数；这些相位函数通常写为 p(cos θ)。具有此类相位函数的介质称为*各向同性*或*对称*介质，因为它们对入射照明的响应在（局部）旋转下是不变的。除了被归一化之外，自然存在的相位函数的一个重要属性是它们是*可逆的*：两个方向可以互换，相位函数的值保持不变。注意对称相位函数天然是可逆的。

In *anisotropic* media that consist of particles arranged in a coherent structure, the phase function can be a 4D function of the two directions, which satisfies a more involved kind of reciprocity relation. Examples of this are crystals or media made of coherently oriented fibers; the "Further Reading" discusses these types of media further.

> 在由排列成相干结构的粒子组成的*各向异性*介质中，相位函数可以是两个方向的四维函数，它满足一种更复杂的互易关系。这方面的例子有晶体或由相干取向纤维组成的介质。

In a slightly confusing overloading of terminology, phase functions themselves can be isotropic or anisotropic as well. Thus, we might have an anisotropic phase function in an isotropic medium. An isotropic phase function describes equal scattering in all directions and is thus independent of either of the two directions. ### 11.3.1 The Henyey–Greenstein Phase Function

> ### 11.3.1 Henyey–Greenstein 相位函数

A widely used phase function was developed by Henyey and Greenstein ([1941](https://pbr-book.org/4ed/Volume_Scattering/Further_Reading.html#cite:Henyey41)). This phase function was specifically designed to be easy to fit to measured scattering data. A single parameter (called the *asymmetry parameter*) controls the distribution of scattered light:

> 一个被广泛使用的相位函数由 Henyey 和 Greenstein (1941) 开发。这个相位函数专门设计为易于拟合测量的散射数据。一个单一参数（称为*不对称参数*）控制散射光的分布。

The HenyeyGreenstein() function implements this computation.

<<Scattering Inline Functions>>+=

Float HenyeyGreenstein(Float cosTheta, Float g) { Float denom = 1 + [Sqr](https://pbr-book.org/4ed/Utilities/Mathematical_Infrastructure.html#Sqr) (g) + 2 \* g \* cosTheta; return [Inv4Pi](https://pbr-book.org/4ed/Utilities/Mathematical_Infrastructure.html#Inv4Pi) \* (1 - [Sqr](https://pbr-book.org/4ed/Utilities/Mathematical_Infrastructure.html#Sqr) (g)) / (denom \* [SafeSqrt](https://pbr-book.org/4ed/Utilities/Mathematical_Infrastructure.html#SafeSqrt) (denom)); }
