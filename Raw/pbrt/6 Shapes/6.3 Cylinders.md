## 6.3 Cylinders

Another useful quadric is the cylinder; pbrt provides a cylinder Shape that is centered around the z axis. The user can supply a minimum and maximum z value for the cylinder, as well as a radius and maximum φ sweep value.

> 另一个有用的二次曲面是圆柱体；pbrt 提供了一个以 z 轴为中心的圆柱体 Shape。用户可以指定圆柱体的最小和最大 z 值，以及半径和最大 φ 扫掠值。

> 与 Sphere 构造函数类似，Cylinder 构造函数接受定义其物体空间的变换以及定义圆柱体本身的参数。它的构造函数只是初始化相应的成员变量，因此我们不在此列出。

### 6.3.1 Area and Bounding

A cylinder is a rolled-up rectangle. If you unroll the rectangle, its height is z_max - z_min, and its width is r × φ_max.

> 圆柱体就是一个卷起来的矩形。如果将矩形展开，其高度为 z_max - z_min，宽度为 r × φ_max。

As was done with the sphere, the cylinder's spatial bounding method computes a conservative bounding box using the z range but does not take into account the maximum φ.

> 与球体一样，圆柱体的空间包围方法使用 z 范围计算保守的包围盒，但不考虑最大 φ。

Its surface normal bounding function is conservative in two ways: not only does it not account for φ_max, but the actual set of normals of a cylinder can be described by a circle on the sphere of all directions. However, DirectionCone's representation is not able to bound such a distribution more tightly than with the entire sphere of directions, and so that is the bound that is returned.

> 其表面法线包围函数在两个方面是保守的：它不仅没有考虑 φ_max，而且圆柱体的实际法线集合可以用方向球面上的一个圆来描述。然而，DirectionCone 的表示无法比整个方向球面更紧密地包围这样的分布，因此返回的就是整个球面。

### 6.3.2 Intersection Tests

Also similar to the sphere (and for similar reasons), Cylinder provides a BasicIntersect() method that returns a QuadricIntersection as well as an InteractionFromIntersection() method that converts that to a full SurfaceInteraction. Given these, the Intersect() method is again a simple composition of them. (If pbrt used virtual functions, a design alternative would be to have a QuadricShape class that provided a default Intersect() method and left BasicIntersect() and InteractionFromIntersection() as pure virtual functions for subclasses to implement.)

> 与球体类似（原因也类似），Cylinder 提供了一个返回 QuadricIntersection 的 BasicIntersect() 方法，以及一个将其转换为完整 SurfaceInteraction 的 InteractionFromIntersection() 方法。有了这些，Intersect() 方法又是它们的简单组合。（如果 pbrt 使用虚函数，一种设计替代方案是拥有一个 QuadricShape 类，提供默认的 Intersect() 方法，并将 BasicIntersect() 和 InteractionFromIntersection() 作为纯虚函数让子类实现。）

The form of the BasicIntersect() method also parallels the sphere's, computing appropriate quadratic coefficients, solving the quadratic equation, and then handling the various cases for partial cylinders. A number of fragments can be reused from the Sphere's implementation.

> BasicIntersect() 方法的形式也与球体的平行，计算适当的二次方程系数、求解二次方程，然后处理部分圆柱体的各种情况。许多代码片段可以从 Sphere 的实现中重用。

As before, the fragment that computes the quadratic discriminant is defined in Section 6.8.3 after topics related to floating-point accuracy have been discussed.

> 与之前一样，计算二次判别式的代码片段在第 6.8.3 节讨论了浮点精度相关主题之后定义。

The ray–cylinder intersection formula can be found by substituting the ray equation into the cylinder's implicit equation. The implicit equation for an infinitely long cylinder centered on the z axis with radius r is x² + y² - r² = 0. Substituting the ray equation, we have the quadratic coefficients.

> 光线-圆柱体交点公式可以通过将光线方程代入圆柱体的隐式方程来求得。以 z 轴为中心、半径为 r 的无限长圆柱体的隐式方程为 x² + y² - r² = 0。代入光线方程后，我们得到二次方程的系数。

As with spheres, the implementation refines the computed intersection point to reduce the rounding error in the point computed by evaluating the ray equation; see Section 6.8.5. Afterward, we invert the parametric description of the cylinder to compute φ from the hit point; it turns out that the result is the same as for the sphere.

> 与球体一样，实现对计算出的交点进行精化，以减少通过求值光线方程计算的点中的舍入误差；见第 6.8.5 节。之后，我们反转圆柱体的参数描述从命中点计算 φ；结果与球体的相同。

The next step in the intersection method makes sure that the hit is in the specified z range and that the φ angle is acceptable. If not, it rejects the hit and checks t1 if it has not already been considered—these tests resemble the conditional logic in Sphere::Intersect().

> 交点方法的下一步确保命中点在指定的 z 范围内，且 φ 角是可接受的。如果不是，它会拒绝该命中并检查 t1（如果尚未考虑过的话）——这些测试类似于 Sphere::Intersect() 中的条件逻辑。

For a successful intersection, the same three values suffice to provide enough information to later compute the corresponding SurfaceInteraction.

> 对于成功的交点，同样的三个值足以提供稍后计算相应 SurfaceInteraction 所需的信息。

As with the sphere, IntersectP()'s implementation is a simple wrapper around BasicIntersect().

> 与球体一样，IntersectP() 的实现是 BasicIntersect() 的简单包装。

InteractionFromIntersection() computes all the quantities needed to initialize a SurfaceInteraction from a cylinder's QuadricIntersection.

> InteractionFromIntersection() 计算从圆柱体的 QuadricIntersection 初始化 SurfaceInteraction 所需的所有量。

Again the parametric u value is computed by scaling φ to lie between 0 and 1. Inversion of the parametric equation for the cylinder's z value gives the v parametric coordinate.

> 同样，参数 u 值通过将 φ 缩放到 0 和 1 之间来计算。反转圆柱体 z 值的参数方程给出 v 参数坐标。

The partial derivatives for a cylinder are easy to derive.

> 圆柱体的偏导数很容易推导。

We again use the Weingarten equations to compute the parametric partial derivatives of the cylinder normal. The relevant partial derivatives are straightforward since d²P/du² = -φ_max² × (pHit.x, pHit.y, 0), d²P/dudv = (0,0,0), and d²P/dv² = (0,0,0).

> 我们再次使用 Weingarten 方程来计算圆柱体法线的参数偏导数。相关的偏导数很直接，因为 d²P/du² = -φ_max² × (pHit.x, pHit.y, 0)，d²P/dudv = (0,0,0)，d²P/dv² = (0,0,0)。

### 6.3.3 Sampling

Uniformly sampling the surface area of a cylinder is straightforward: uniform sampling of the height and φ give uniform area sampling. Intuitively, it can be understood that this approach works because a cylinder is just a rolled-up rectangle.

> 均匀采样圆柱体的表面积很简单：均匀采样高度和 φ 就能给出均匀面积采样。直觉上可以理解这种方法之所以有效，是因为圆柱体就是一个卷起来的矩形。

Given z and φ, the corresponding object-space position and normal are easily found.

> 给定 z 和 φ，相应的物体空间位置和法线很容易求得。

Unlike the Sphere, pbrt's Cylinder does not have a specialized solid angle sampling method. Instead, it samples a point on the cylinder uniformly by area without making use of the reference point before converting the area density for that point to a solid angle density before returning it. Both the Sample() and PDF() methods can be implemented using the same fragments that were used for solid angle sampling of reference points inside spheres.

> 与 Sphere 不同，pbrt 的 Cylinder 没有专门的立体角采样方法。相反，它通过面积均匀地在圆柱体上采样一个点，不利用参考点，然后在返回之前将该点的面积密度转换为立体角密度。Sample() 和 PDF() 方法都可以使用与球体内部参考点的立体角采样相同的代码片段来实现。
