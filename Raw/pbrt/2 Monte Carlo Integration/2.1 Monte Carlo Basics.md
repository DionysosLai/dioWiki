## 2.1 Monte Carlo: Basics

Because Monte Carlo integration is based on randomization, we will start this chapter with a brief review of ideas from probability and statistics that provide the foundations of the approach.

> 由于蒙特卡洛积分基于随机化，我们将以概率和统计中为该方法提供基础的思想的简要回顾开始本章。

### 2.1.1 Background and Probability Review

> ### 2.1.1 背景与概率回顾

A *random variable* is a value chosen by some random process, drawn from some domain (discrete or continuous). A uniform random variable has the same probability for all potential values. A *probability mass function* (PMF) gives a discrete random variable's probability.

> *随机变量*是由某种随机过程从某个域（离散或连续）中选择的值。均匀随机变量对所有可能值具有相同概率。*概率质量函数*（PMF）给出离散随机变量的概率。

Two random variables are *independent* if one's probability does not affect the other's. For *dependent* variables, the joint probability is p(x,y) = p(x) · p(y|x), where p(y|x) is the *conditional probability*.

> 两个随机变量是*独立*的，如果一个的概率不影响另一个。对于*相依*变量，联合概率为 p(x,y) = p(x) · p(y|x)，其中 p(y|x) 是*条件概率*。

The *canonical uniform random variable* ξ takes all values in [0,1) with uniform probability. It is important because it is easy to generate and can be mapped to any discrete distribution.

> *标准均匀随机变量* ξ 以均匀概率取 [0,1) 中的所有值。它之所以重要，因为容易生成且可映射到任何离散分布。

The *CDF* P(x) = Pr(X ≤ x). The *PDF* p(x) = dP(x)/dx is nonneg and integrates to 1.

> *CDF* P(x) = Pr(X ≤ x)。*PDF* p(x) = dP(x)/dx 非负且积分为 1。

### 2.1.2 Expected Values

E[f(x)] = ∫ f(x) p(x) dx. Properties: E[af(x)] = aE[f(x)], E[∑f] = ∑E[f].

> ### 2.1.2 期望值
>
> E[f(x)] = ∫ f(x) p(x) dx。性质：E[af(x)] = aE[f(x)]，E[∑f] = ∑E[f]。

### 2.1.3 The Monte Carlo Estimator

For uniform Xᵢ ∈ [a,b]: Fₙ = (b-a)/n · ∑ f(Xᵢ). For general PDF p: Fₙ = (1/n) ∑ f(Xᵢ)/p(Xᵢ). The 4π factor in RandomWalkIntegrator comes from uniform sphere sampling (area 4π, PDF = 1/(4π)).

> ### 2.1.3 蒙特卡洛估计量
>
> 对于均匀 Xᵢ ∈ [a,b]：Fₙ = (b-a)/n · ∑ f(Xᵢ)。对于一般 PDF p：Fₙ = (1/n) ∑ f(Xᵢ)/p(Xᵢ)。RandomWalkIntegrator 中的 4π 因子来自均匀球面采样（面积 4π，PDF = 1/(4π)）。

### 2.1.4 Error in Monte Carlo Estimators

V[X] = E[X²] - E[X]². Variance decreases as O(1/n), error as O(1/√n). Monte Carlo's convergence is dimension-independent, unlike quadrature. Efficiency = 1/(V·T). *Biased* estimators that converge as n→∞ are *consistent*. MSE = V + bias². *Sample variance* uses Bessel's correction: dividing by n-1.

> ### 2.1.4 蒙特卡洛估计量中的误差
>
> V[X] = E[X²] - E[X]²。方差以 O(1/n) 下降，误差以 O(1/√n) 下降。蒙特卡洛的收敛与维度无关，不同于求积。效率 = 1/(V·T)。随着 n→∞ 收敛的*有偏*估计量是*一致的*。MSE = V + 偏差²。*样本方差*使用 Bessel 校正：除以 n-1。
