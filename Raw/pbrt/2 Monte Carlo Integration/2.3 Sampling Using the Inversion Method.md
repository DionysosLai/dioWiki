## 2.3 Sampling Using the Inversion Method

To evaluate the Monte Carlo estimator, it is necessary to draw random samples from a chosen probability distribution. The *inversion method* maps uniform samples from [0,1) to a given 1D probability distribution by inverting the distribution's CDF. Throughout the book, we will see its application to BSDFs, light sources, cameras, and scattering media.

> 为了计算蒙特卡洛估计量，需要从选定的概率分布中抽取随机样本。*反转方法*通过反转分布的 CDF，将 [0,1) 中的均匀样本映射到给定的一维概率分布。在全书中，我们将看到它在 BSDF、光源、相机和散射介质中的应用。

### 2.3.1 Discrete Case

For a set of discrete probabilities, the CDF is built by stacking the PMF bars. Sampling corresponds to finding i such that P(i-1) < ξ ≤ P(i), which is equivalent to inverting the CDF. The SampleDiscrete() function implements this.

> ### 2.3.1 离散情况
>
> 对于一组离散概率，CDF 通过堆叠 PMF 条形图构建。采样对应于找到 i 使得 P(i-1) < ξ ≤ P(i)，这等价于反转 CDF。SampleDiscrete() 函数实现了这一点。

```cpp
int SampleDiscrete(pstd::span<const Float> weights, Float u,
                   Float *pmf, Float *uRemapped) {
    if (weights.empty()) { if (pmf) *pmf = 0; return -1; }
    Float sumWeights = 0;
    for (Float w : weights) sumWeights += w;
    Float up = u * sumWeights;
    if (up == sumWeights) up = NextFloatDown(up);
    int offset = 0; Float sum = 0;
    while (sum + weights[offset] <= up) sum += weights[offset++];
    if (pmf) *pmf = weights[offset] / sumWeights;
    if (uRemapped) *uRemapped = std::min((up - sum) / weights[offset],
                                          OneMinusEpsilon);
    return offset;
}
```

> SampleDiscrete() 接受一组（不一定归一化的）非负权重和一个均匀随机样本 u，返回权重之一的索引，其概率与权重成正比。它还可以返回该样本的 PMF 值以及从 u 派生的新均匀随机样本。

### 2.3.2 Continuous Case

The continuous inversion method: (1) integrate the PDF to find the CDF, (2) obtain a uniform random number ξ, (3) solve P(x) = ξ for x, i.e., find x = P⁻¹(ξ).

> ### 2.3.2 连续情况
>
> 连续反转方法：（1）对 PDF 积分得到 CDF，（2）获取均匀随机数 ξ，（3）求解 P(x) = ξ 得到 x，即 x = P⁻¹(ξ)。

#### Sampling a Linear Function

The function `Lerp(x, a, b) = (1-x)·a + x·b` over [0,1] has integral (a+b)/2, giving PDF = 2·Lerp(x,a,b)/(a+b). Integrating the PDF gives a quadratic CDF, which can be inverted analytically:

> #### 采样线性函数
>
> 函数 `Lerp(x, a, b) = (1-x)·a + x·b` 在 [0,1] 上的积分为 (a+b)/2，给出 PDF = 2·Lerp(x,a,b)/(a+b)。对 PDF 积分得到二次 CDF，可以解析反转：

```cpp
Float SampleLinear(Float u, Float a, Float b) {
    if (u == 0 && a == 0) return 0;
    Float x = u * (a + b) / (a + std::sqrt(Lerp(u, Sqr(a), Sqr(b))));
    return std::min(x, OneMinusEpsilon);
}
```

```cpp
Float InvertLinearSample(Float x, Float a, Float b) {
    return x * (a * (2 - x) + b * x) / (a + b);
}
```
