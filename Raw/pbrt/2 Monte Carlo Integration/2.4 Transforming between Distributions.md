## 2.4 Transforming between Distributions

We investigate the general question of which distribution results when we transform samples from an arbitrary distribution with a function y = T(x). Understanding these transformations allows us to derive multidimensional sampling algorithms.

> 我们研究一个一般性问题：当我们用函数 y = T(x) 变换来自任意分布的样本时会产生什么分布。理解这些变换使我们能够推导多维采样算法。

Given random variable X with PDF pₓ and a one-to-one function T, the PDF of Y = T(X) is:

p_y(y) = p_x(T⁻¹(y)) / |dT/dx|

> 给定具有 PDF pₓ 的随机变量 X 和一一映射函数 T，Y = T(X) 的 PDF 为：
>
> p_y(y) = p_x(T⁻¹(y)) / |dT/dx|

### 2.4.1 Transformation in Multiple Dimensions

In the general D-dimensional case, the densities are related by:

p_y(y) = p_x(T⁻¹(y)) / |J_T|

where |J_T| is the absolute value of the determinant of T's Jacobian matrix.

> ### 2.4.1 多维变换
>
> 在一般 D 维情况下，密度之间的关系为：
>
> p_y(y) = p_x(T⁻¹(y)) / |J_T|
>
> 其中 |J_T| 是 T 的雅可比矩阵行列式的绝对值。

**Polar coordinates**: The Jacobian determinant of the polar transformation is r, so p(r,θ) = r · p(x,y).

> **极坐标**：极坐标变换的雅可比行列式为 r，因此 p(r,θ) = r · p(x,y)。

**Spherical coordinates**: The Jacobian determinant is sin θ, so p(θ,φ) = sin θ · p(ω). This is important for representing directions as points on the unit sphere.

> **球坐标**：雅可比行列式为 sin θ，因此 p(θ,φ) = sin θ · p(ω)。这对于将方向表示为单位球面上的点很重要。

### 2.4.2 Sampling with Multidimensional Transformations

For a 2D joint density p(x,y), the *marginal density* is p(x) = ∫ p(x,y) dy. The *conditional density* is p(y|x) = p(x,y)/p(x). Sample x from the marginal, then y from the conditional.

> ### 2.4.2 用多维变换采样
>
> 对于二维联合密度 p(x,y)，*边际密度*为 p(x) = ∫ p(x,y) dy。*条件密度*为 p(y|x) = p(x,y)/p(x)。从边际分布采样 x，然后从条件分布采样 y。

#### Sampling the Bilinear Function

The bilinear function interpolates four corner values w₀, w₁, w₂, w₃ over [0,1]². Since the two dimensions are not independent, sampling uses a marginal distribution first (for y), then a conditional distribution (for x), both using SampleLinear().

> #### 采样双线性函数
>
> 双线性函数在 [0,1]² 上插值四个角值 w₀、w₁、w₂、w₃。由于两个维度不独立，采样首先使用边际分布（对 y），然后使用条件分布（对 x），两者都使用 SampleLinear()。

```cpp
Point2f SampleBilinear(Point2f u, pstd::span<const Float> w) {
    Point2f p;
    p.y = SampleLinear(u[1], w[0] + w[1], w[2] + w[3]);
    p.x = SampleLinear(u[0], Lerp(p.y, w[0], w[2]),
                              Lerp(p.y, w[1], w[3]));
    return p;
}
```
