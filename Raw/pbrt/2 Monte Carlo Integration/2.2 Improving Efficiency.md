## 2.2 Improving Efficiency

Given an unbiased Monte Carlo estimator, we have a reliable relationship between the number of samples and variance. If we have an unacceptably noisy image, increasing samples reduces error predictably. However, computation takes time, so we need techniques to make more of the samples we can take.

> 给定一个无偏蒙特卡洛估计量，我们在样本数和方差之间有可靠的关系。如果图像噪声不可接受，增加样本数可预测地减少误差。然而，计算需要时间，因此我们需要技术来更好地利用能够取的样本。

### 2.2.1 Stratified Sampling

Stratified sampling decomposes the integration domain into non-overlapping regions (strata) and places samples in each one. It can never increase variance — it always reduces it unless the function has the same mean over each stratum. For best results, strata should have means as unequal as possible. Compact strata are desirable when the function is unknown.

> ### 2.2.1 分层采样
>
> 分层采样将积分域分解为不重叠的区域（层），并在每个区域中放置样本。它永远不会增加方差——它总是减少方差，除非函数在每个层上具有相同的均值。为获得最佳效果，各层应具有尽可能不等的均值。当函数未知时，紧凑的层是可取的。

The main downside is the "curse of dimensionality": full stratification in D dimensions with S strata per dimension requires S^D samples. Fortunately, it is often possible to stratify some dimensions independently and randomly associate samples from different dimensions.

> 主要缺点是"维度诅咒"：在 D 维中每维 S 个层的完全分层需要 S^D 个样本。幸运的是，通常可以独立分层某些维度并随机关联来自不同维度的样本。

### 2.2.2 Importance Sampling

Importance sampling exploits the fact that the Monte Carlo estimator converges more quickly if samples are drawn from a distribution similar to the integrand. Samples are more likely taken where the integrand's magnitude is large.

> ### 2.2.2 重要性采样
>
> 重要性采样利用了这样一个事实：如果样本从与被积函数相似的分布中抽取，蒙特卡洛估计量会更快收敛。样本更可能在被积函数幅度较大的地方被取到。

If we could sample from p(x) ∝ f(x), variance would be zero — but that requires knowing the integral already. In practice, finding a density similar in shape to f reduces variance significantly. A poorly chosen distribution can increase variance dramatically.

> 如果我们能从 p(x) ∝ f(x) 采样，方差将为零——但这需要已经知道积分值。在实践中，找到与 f 形状相似的密度可以显著减少方差。选择不当的分布可能大幅增加方差。

### 2.2.3 Multiple Importance Sampling

We are frequently faced with integrals of products f(x) = g(x)·h(x), where we can importance-sample each factor separately but not their product. MIS addresses this with an easy-to-implement variance reduction technique.

> ### 2.2.3 多重重要性采样
>
> 我们经常面临乘积形式的积分 f(x) = g(x)·h(x)，我们可以分别对每个因子进行重要性采样但不能对其乘积采样。MIS 用一种易于实现的方差减少技术解决了这个问题。

The **balance heuristic** weights each sample by dividing f(x)/pᵢ(x) by the sum of all PDFs, rather than just the one that generated the sample. This downweights samples that one technique generates with low probability at points where another technique has higher probability.

> **平衡启发式**通过将 f(x)/pᵢ(x) 除以所有 PDF 之和（而非仅生成该样本的那个 PDF）来加权每个样本。这降低了某一技术在另一技术具有更高概率的点以低概率生成的样本的贡献。

```cpp
Float BalanceHeuristic(int nf, Float fPdf, int ng, Float gPdf) {
    return (nf * fPdf) / (nf * fPdf + ng * gPdf);
}
```

The **power heuristic** (with β=2) often reduces variance even further:

> **幂启发式**（β=2）通常进一步减少方差：

```cpp
Float PowerHeuristic(int nf, Float fPdf, int ng, Float gPdf) {
    Float f = nf * fPdf, g = ng * gPdf;
    return Sqr(f) / (Sqr(f) + Sqr(g));
}
```

**MIS Compensation** can further reduce variance by sharpening distributions to have zero probability where they had low probability, effectively concentrating sampling where it matters most.

> **MIS 补偿**可以通过锐化分布——在先前具有低概率的区域设为零概率——来进一步减少方差，有效地将采样集中在最重要的地方。

### 2.2.4 Russian Roulette

Russian roulette improves efficiency by skipping evaluation of samples that would make a small contribution. Given termination probability q, with probability q the estimator is replaced by constant c (often 0); with probability 1-q, the estimator is weighted by 1/(1-q) to compensate. The expected value remains unchanged. Russian roulette never reduces variance but improves efficiency.

> ### 2.2.4 俄罗斯轮盘赌
>
> 俄罗斯轮盘赌通过跳过对贡献很小的样本的评估来提高效率。给定终止概率 q，以概率 q 用常量 c（通常为 0）替换估计量；以概率 1-q，估计量乘以 1/(1-q) 作为补偿。期望值保持不变。俄罗斯轮盘赌永远不会减少方差但会提高效率。

### 2.2.5 Splitting

While Russian roulette reduces samples, splitting increases samples in some dimensions of multidimensional integrals. For example, in rendering, an integral over pixel area contains for each point a hemisphere integral for lighting — splitting takes multiple lighting samples per camera ray, amortizing the cost of tracing the initial ray.

> ### 2.2.5 分裂
>
> 俄罗斯轮盘赌减少样本，而分裂在多维积分的某些维度上增加样本。例如在渲染中，像素面积上的积分中每个点包含一个用于光照的半球积分——分裂为每条相机光线取多个光照样本，分摊追踪初始光线的成本。
