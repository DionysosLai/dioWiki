## 8.7 Sobol' Samplers

While the Halton sequence is effective for Monte Carlo integration, each radical inverse computation requires one integer division per digit. The integer division instruction is one of the slowest on most processors. Therefore, this section describes three Samplers based on the Sobol' sequence, a low-discrepancy sequence defined entirely in base 2, which leads to efficient implementations.

> 虽然 Halton 序列对蒙特卡洛积分有效，但每次基数反转计算需要每位数字一次整数除法。整数除法指令是大多数处理器上最慢的指令之一。因此，本节描述了三种基于 Sobol' 序列的 Sampler，这是一个完全在基数 2 中定义的低偏差序列，能实现高效实现。

The base-2 radical inverse can be computed efficiently by reversing the bits of the integer value and multiplying by 2^{-64}. The Sobol' sequence uses base 2 for all dimensions, with a different *generator matrix* C for each dimension. The sample point is computed by the matrix-vector product C · a (mod 2) (Equation 8.22), where all arithmetic is in Z_2.

> 基数 2 的基数反转可以通过反转整数值的位并乘以 2^{-64} 来高效计算。Sobol' 序列对所有维度使用基数 2，每个维度使用不同的**生成矩阵** C。采样点通过矩阵-向量乘积 C · a (mod 2)（公式 8.22）计算，所有运算在 Z_2 中进行。

In base 2, matrix entries are 0 or 1, so each column can be represented as a uint32_t. The matrix-vector product (Equation 8.23) reduces to XOR-ing together the columns of C where a's corresponding bit is 1—implemented efficiently by `MultiplyGenerator()`. If the identity matrix is used, this gives the regular van der Corput sequence.

> 在基数 2 中，矩阵元素为 0 或 1，因此每列可以用一个 uint32_t 表示。矩阵-向量乘积（公式 8.23）简化为对 a 对应位为 1 的 C 的列进行异或——由 `MultiplyGenerator()` 高效实现。如果使用单位矩阵，则给出常规的 van der Corput 序列。

### 8.7.1 Stratification over Elementary Intervals

The first two dimensions of the Sobol' sequence are stratified in a very general way. For example, the first 16 samples satisfy the 4×4 stratification constraint, and are also stratified over all 2×8, 8×2, 1×16, and 16×1 boxes. Any subsequent power-of-2-sized set of samples fulfills the same constraints. Such a sequence is called a *(0,2)-sequence*.

> Sobol' 序列的前两个维度以非常一般的方式分层。例如，前 16 个样本满足 4×4 分层约束，也在所有 2×8、8×2、1×16 和 16×1 框上分层。任何后续的 2 的幂大小的样本集都满足相同的约束。这样的序列称为 **(0,2)-序列**。

The set of *elementary intervals* in two dimensions, base 2, is defined by boxes where s_1 + s_2 = log_2(N). One sample from each of the first N values will be in each elementary interval, and the same is true for each subsequent set of N values.

> 二维基数 2 的**初等区间**集合由 s_1 + s_2 = log_2(N) 的盒子定义。前 N 个值中每个初等区间恰好有一个样本，每个后续的 N 个值集合也是如此。

### 8.7.2 Randomization and Scrambling

For the same reasons as with the Halton sequence, scrambling the Sobol' sequence is useful. Base-2 representation allows efficient scrambling algorithms:

> 出于与 Halton 序列相同的原因，对 Sobol' 序列进行置乱是有用的。基数 2 表示允许高效的置乱算法：

- **NoRandomizer**: Returns the value unchanged.
- **BinaryPermuteScrambler**: XORs with a random value—equivalent to a single random digit permutation applied to all digits, randomly flipping each bit.
- **FastOwenScrambler**: Approximates Owen scrambling by hashing with `MixBits()` and reversing bits. Efficient on GPUs but only an approximation.
- **OwenScrambler**: Full Owen scrambling, processing each bit from MSB to LSB, XOR-ing with a hash of higher bits. Best quality.

> - **NoRandomizer**：不做改变直接返回值。
> - **BinaryPermuteScrambler**：与随机值异或——等价于对所有数字应用单一随机排列，随机翻转每一位。
> - **FastOwenScrambler**：通过 `MixBits()` 哈希和位反转近似 Owen 置乱。GPU 上高效但只是近似。
> - **OwenScrambler**：完整 Owen 置乱，从最高位到最低位处理每一位，与高位哈希异或。质量最好。

### 8.7.3 Sobol' Sample Generation

The `SobolSample()` function generates a Sobol' sample for a given index and dimension, applying the provided randomizer. It is templated on the randomizer type for performance. The computation XORs together relevant columns of `SobolMatrices32[]`. pbrt supports NSobolDimensions = 1024 dimensions with SobolMatrixSize = 52 columns per matrix.

> `SobolSample()` 函数为给定索引和维度生成 Sobol' 采样，应用提供的随机化器。它对随机化器类型进行模板化以提高性能。计算异或 `SobolMatrices32[]` 的相关列。pbrt 支持 NSobolDimensions = 1024 个维度，每个矩阵 SobolMatrixSize = 52 列。

### 8.7.4 Global Sobol' Sampler

The SobolSampler uniformly scales the first two dimensions by the smallest power of 2 covering the image area. `SobolIntervalToIndex()` maps pixel coordinates and sample index to the global Sobol' sequence index—scaling means the leading bits select pixels, and the generator matrix inverse allows computing which index lands in a given pixel.

> SobolSampler 用覆盖图像区域的最小 2 的幂均匀缩放前两个维度。`SobolIntervalToIndex()` 将像素坐标和采样索引映射到全局 Sobol' 序列索引——缩放意味着前导位选择像素，生成矩阵的逆允许计算哪个索引落在给定像素中。

All four randomization strategies are supported. `GetPixel2D()` uses unscrambled samples for the first two dimensions, then remaps to within-pixel coordinates.

> 所有四种随机化策略都受支持。`GetPixel2D()` 对前两个维度使用未置乱的样本，然后重新映射到像素内坐标。

### 8.7.5 Padded Sobol' Sampler

The SobolSampler generates points with low discrepancy over all dimensions, but 2D slices (e.g., dimensions 4 and 5) may not be well distributed. This can cause poorly distributed samples over light sources, manifesting as characteristic checkerboard patterns at low sampling rates.

> SobolSampler 生成在所有维度上低偏差的点，但二维切片（如维度 4 和 5）可能分布不佳。这可能导致光源上采样分布不良，在低采样率下表现为特征性棋盘格模式。

The PaddedSobolSampler focuses on good distributions for each 1D/2D sample independently via padding. Each `Get1D()` or `Get2D()` uses a hash of pixel, sample index, dimension, and seed to determine the Sobol' index, then generates only dimensions 0 (1D) or 0–1 (2D). This ensures well-stratified 2D projections but not simultaneous low discrepancy across all dimensions.

> PaddedSobolSampler 着重于通过填充为每个 1D/2D 采样独立提供良好分布。每次 `Get1D()` 或 `Get2D()` 使用像素、采样索引、维度和种子的哈希确定 Sobol' 索引，然后只生成维度 0（1D）或 0–1（2D）。这确保了良好分层的二维投影，但不能在所有维度上同时保持低偏差。

### 8.7.6 Blue Noise Sobol' Sampler

The ZSobolSampler generates blue noise samples by exploiting (0,2)-sequence properties. Key idea: if all pixels take different power-of-2-aligned segments from a single large Sobol' set, nearby pixels use different sample values—this is blue noise decorrelation.

> ZSobolSampler 通过利用 (0,2)-序列性质生成蓝噪声样本。关键思想：如果所有像素从单一大型 Sobol' 集中取不同的 2 的幂对齐段，相邻像素使用不同的采样值——这就是蓝噪声去相关性。

The Morton curve provides the mechanism: computing each pixel's Morton code determines its starting index into the Sobol' set, ensuring nearby pixels (in both x and y) use nearby but distinct segments. `StartPixelSample()` constructs the initial index from the Morton code plus the sample index via left shift. `GetSampleIndex()` randomizes this using Owen scrambling in base 4—permuting each pair of bits based on higher bits, preserving the (0,2)-sequence property while decorrelating samples.

> Morton 曲线提供了机制：计算每个像素的 Morton 码确定其在 Sobol' 集中的起始索引，确保相邻像素（在 x 和 y 方向上）使用相邻但不同的段。`StartPixelSample()` 从 Morton 码加上通过左移的采样索引构造初始索引。`GetSampleIndex()` 使用基数 4 的 Owen 置乱来随机化——根据高位排列每对位，在保持 (0,2)-序列性质的同时去除样本相关性。

### 8.7.7 Evaluation

The SobolSampler often has the lowest numeric error due to global low-discrepancy properties, but structured error can be visually unpleasant. PaddedSobolSampler provides better 2D projections and less structured error. ZSobolSampler combines low error with blue noise visual benefit.

> SobolSampler 由于全局低偏差特性通常数值误差最低，但结构化误差可能视觉上不悦目。PaddedSobolSampler 提供更好的二维投影和更少的结构化误差。ZSobolSampler 结合了低误差和蓝噪声视觉优势。

For defocus blur scenes: Halton has higher error than Stratified (poor 2D lens projections), PaddedSobol gives little benefit over stratified, SobolSampler has remarkably low error despite visible structure, and ZSobolSampler combines low error with blue noise distribution.

> 对于散焦模糊场景：Halton 误差高于分层采样（镜头二维投影差），PaddedSobol 比分层没有太大优势，SobolSampler 尽管有可见结构但误差极低，ZSobolSampler 结合了低误差和蓝噪声分布。

For simple 2D test functions, Sobol' sampling with Owen scrambling achieves asymptotically faster convergence for smooth functions. For the complex blurry dragon test scene, Halton and Sobol' samplers have roughly 10% lower MSE than independent sampling.

> 对于简单二维测试函数，Owen 置乱的 Sobol' 采样对光滑函数实现了渐近更快的收敛。对于复杂的模糊龙测试场景，Halton 和 Sobol' 采样器比独立采样 MSE 低约 10%。
