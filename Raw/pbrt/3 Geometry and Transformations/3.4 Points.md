## 3.4 Points

A point is a zero-dimensional location in 2D or 3D space. The Point2 and [Point3](#Point3) classes in pbrt represent points in the obvious way: using,, (in 3D) coordinates with respect to a coordinate system. Although the same representation is used for vectors, the fact that a point represents a position whereas a vector represents a direction leads to a number of important differences in how they are treated. Points are denoted in text by.

> 点是二维或三维空间中的零维位置。pbrt 中的 Point2 和 Point3 类以显而易见的方式表示点：使用相对于坐标系的 x、y（以及三维中的 z）坐标。虽然向量使用相同的表示，但点表示位置而向量表示方向这一事实导致了它们在处理方式上的若干重要差异。文中用 p 来表示点。

In this section, we will continue the approach of only including implementations of the 3D point methods for the Point3 class here.

> 在本节中，我们将继续采用仅包含 Point3 类的三维点方法实现的做法。

As with vectors, it is helpful to have shorter type names for commonly used point types.

> 与向量一样，为常用的点类型提供更短的类型名称很有帮助。

<<Point3\* Definitions>>=

using Point3f = Point3<Float>; using Point3i = Point3<int>;

It is also useful to be able to convert a point with one element type (e.g., a Point3f) to a point of another one (e.g., Point3i) as well as to be able to convert a point to a vector with a different underlying element type. The following constructor and conversion operator provide these conversions. Both also require an explicit cast, to make it clear in source code when they are being used.

> 能够将一种元素类型的点（如 Point3f）转换为另一种类型的点（如 Point3i），以及能够将点转换为具有不同底层元素类型的向量，也很有用。以下构造函数和转换运算符提供了这些转换。两者都需要显式转换，以便在源代码中明确何时使用它们。

There are certain [Point3](#Point3) methods that either return or take a [Vector3](https://pbr-book.org/4ed/Geometry_and_Transformations/Vectors.html#Vector3). For instance, one can add a vector to a point, offsetting it in the given direction to obtain a new point. Analogous methods, not included in the text, also allow subtracting a vector from a point.

> 某些 Point3 方法要么返回要么接受 Vector3。例如，可以将一个向量加到一个点上，沿给定方向偏移以获得一个新点。文中未包含的类似方法也允许从点中减去向量。

Alternately, one can subtract one point from another, obtaining the vector between them, as shown in Figure [3.7](#fig:pointsub).

> 另外，可以将一个点减去另一个点，获得它们之间的向量，如图 3.7 所示。

[![](https://pbr-book.org/4ed/Geometry_and_Transformations/pha03f07.svg)](https://pbr-book.org/4ed/Geometry_and_Transformations/pha03f07.svg)

Figure 3.7: Obtaining the Vector between Two Points. The vector is given by the component-wise subtraction of the points and.

> 图 3.7：获取两点之间的向量。该向量由两个点的逐分量相减给出。

The distance between two points can be computed by subtracting them to compute the vector between them and then finding the length of that vector. Note that we can just use auto for the return type and let it be set according to the return type of Length(); there is no need to use the [TupleLength](https://pbr-book.org/4ed/Geometry_and_Transformations/Vectors.html#TupleLength) type trait to find that type.

> 两点之间的距离可以通过将它们相减计算出之间的向量，然后求该向量的长度来得到。注意我们可以直接用 auto 作为返回类型，让它根据 Length() 的返回类型来设定；无需使用 TupleLength 类型特征来确定该类型。

<<Point3 Inline Functions>>=

template <typename T> auto Distance(Point3<T> p1, Point3<T> p2) { return [Length](https://pbr-book.org/4ed/Geometry_and_Transformations/Vectors.html#Length) (p1 - p2); }

The squared distance between two points can be similarly computed using LengthSquared().

> 两点之间距离的平方可以类似地使用 LengthSquared() 来计算。

<<Point3 Inline Functions>>+=

template <typename T> auto DistanceSquared(Point3<T> p1, Point3<T> p2) { return [LengthSquared](https://pbr-book.org/4ed/Geometry_and_Transformations/Vectors.html#LengthSquared) (p1 - p2); }
